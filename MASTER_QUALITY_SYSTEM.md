## PHASE 1: RESEARCH & VERIFICATION (4-5 hours)

### Step 1: Verify Device Exists (30 minutes)
```
□ Check official brand website
  - vivo.com/in / samsung.com / oneplus.com / etc.
  - Search for exact model
  - Screenshot the official page

□ Check GSMArena
  - gsmarena.com
  - Search exact phone model
  - Compare specs with official source
  - Note any discrepancies

□ Check Indian retailers
  - Amazon.in
  - Flipkart
  - Note: In stock / price

Result: ✅ Confirmed real phone / ❌ Phone doesn't exist
```

### Step 2: Gather All Official Specifications (1 hour)
```
Create a spreadsheet with:
- Model name
- Processor (exact name & version)
- RAM options
- Storage options
- Display specs (size, type, refresh rate, brightness)
- Camera specs (all lenses, MP, aperture, focal length)
- Battery (capacity, type, charging wattage)
- Charging time (official claim)
- Colors available
- Dimensions & weight (exact)
- IP rating
- Update promise (years)
- Price (all variants, current)

For each spec:
□ Source: Official website / GSMArena / Retailer
□ Verification date: [Date]
□ Screenshot saved: YES / NO
```

**Sources to use (in order of priority):**
1. Official manufacturer website
2. Official press release
3. GSMArena (verified database)
4. Government certification (BIS/FCC)
5. Retailer official listing

### Step 3: Verify Pricing (30 minutes)
```
For EACH variant (8GB, 12GB, 16GB etc.):

□ Bank offers: _______________
□ Verification date & time: _______________
□ Verified by screenshot: YES / NO

Note: Prices change frequently
Always verify within 1-2 weeks of publishing
```

## ❌ ERRORS TO AVOID (At All Costs)

### 🚩 ERROR 1: Publishing without verifying phone exists
**Impact:** Article about non-existent product
**Solution:** Check 3 sources: Official site, GSMArena, Retailers

### 🚩 ERROR 2: Outdated pricing
**Impact:** Reader buys at wrong price
**Solution:** Verify prices monthly, add date stamps

### 🚩 ERROR 3: False update promises
**Impact:** Reader regrets purchase (expects updates that won't come)
**Solution:** Verify only from official support pages

### 🚩 ERROR 4: Unverified performance claims
**Impact:** Reader expectations don't match reality
**Solution:** Disclose testing methodology or use official specs

### 🚩 ERROR 5: Copying from competitors
**Impact:** Plagiarism issues, AdSense rejection
**Solution:** Use official sources, write original comparisons

### 🚩 ERROR 6: Unfair comparisons
**Impact:** Reader distrust, legal issues
**Solution:** Use current specs, acknowledge competitor strengths

### 🚩 ERROR 7: No source attribution
**Impact:** Readers can't verify claims
**Solution:** Add citations for major claims
