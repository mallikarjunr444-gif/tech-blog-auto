## ⚠️ CRITICAL RULES FOR ADAPTATION

### ✅ MUST DO:

1. **Verify every specification**
   - Don't assume specs are accurate
   - Check official source
   - Cross-check with GSMArena
   - Document source

2. **Update prices to current**
   - Prices change frequently
   - Verify within 1-2 weeks of publishing
   - Add date stamp: "As of [date]"
   - Add disclaimer: "Prices may have changed"

3. **Verify update promises**
   - Contact manufacturer if unclear
   - Use only official statements
   - Include source URL
   - This claim affects buying decisions

4. **Fair comparisons**
   - Don't disparage competitors
   - Use actual specs, not assumptions
   - Acknowledge competitor strengths
   - Use current models, not outdated ones

5. **Source attribution**
   - Cite major claims
   - Link to sources where possible
   - Document verification dates
   - Disclose testing methodology

---

### ❌ NEVER DO:

1. ❌ **Publish without verifying phone exists**
   - Check multiple sources
   - Confirm specs match
   - Don't use fictional phones

2. ❌ **Use outdated prices**
   - Prices change monthly
   - Update before publishing
   - Add date stamps

3. ❌ **Make untested claims**
   - Say "According to official specs" instead
   - Or actually test and document
   - Don't claim testing you didn't do

4. ❌ **Copy specs from competitors**
   - Could be plagiarism
   - Use official sources instead
   - Or create original comparison

5. ❌ **Forget about update promises**
   - Verify these claims carefully
   - Readers depend on this for long-term buying decisions
   - Source from official support pages
